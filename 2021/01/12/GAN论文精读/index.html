学习论文：Generative Adversarial Nets

论文目的：Although the discriminative model in deep learning has make a great progress ,the generative model have had less of an impact . So this paper make a import breakthtough in genetative model .

论文摘要：He propose a new framewordk called GAN(Generative Adversarial Nets).It trains two model : a generative model G to generate the distribution of data , a discriminative model D to estimate whether the input is fake.So the G’s goal is maximize the error of D,the D’s goal is minimize the error of decision.So the pefect situation is D’s error rate is 50% because the G and the ground truth have the same distribution.

关键词：generative model

related work

Before this work ,most work on deep generative models focus on models that provided a parametric specification of a probability distribution function,such as the deep Boltzmann machine and Generative stochastic networks .The model can be trained by maximizing the log likelihood.(This method need a lot of mathematic skills ,so I don’t learn).

It also introduce VAE.(I don’t know)

It has also use a discriminative criterion to train a generative model,such as NCE(Obviously ,I don’t know)

Some previous work has used the general concept of having two neural networks compete.（Interesing），such as predictability minimization.In predictability minimization, each hidden unit in a neural network is trained to be different from the output of a second network, which predicts the value of that hidden unit given the value of all of the other hidden units.(But in fact ,I can’t understand its meaning.)

可能大概意思是两个竞争网络一个想尽可能的贴近第二个的输出，第二个是想尽可能输出不像的东西，与GAN不同的地方就在于它是一个优化问题，而GAN则更类似于一个minmax game。

他还介绍了Adversial example，它使得设计出来对抗使得神经网络错误分类的。这有着重要意义。（后面说的那句话我没看懂，但无伤大雅。）

总体训练过程

用BCELoss，这和交叉熵有什么区别吗？

BCELoss是CrossEntropyLoss的一个特例，只用于二分类问题，而CrossEntropyLoss可以用于二分类，也可以用于多分类。不过我重新查阅了一下资料，发现同样是处理二分类问题，BCELoss与CrossEntropyLoss是不同的。下面我详细讲一下哪里不同。

１．使用nn.BCELoss需要在该层前面加上Sigmoid函数。
公式如下：
$l o s s ( X_i , y_ i ) = − w_i [ y_i l o g x_i + ( 1 − y_i ) l o g ( 1 − x_i ) ] $
２．　使用nn.CrossEntropyLoss会自动加上Sofrmax层。这里就不详细介绍了

可以看出，这两个计算损失的函数使用的激活函数不同，故而最后的计算公式不同。

总结

作为GAN论文大开篇之作，不得不说，这里有很多槽点，但是不可否认的是，这里面也蕴含着巨大的潜力。

正如论文中所说的conditional GAN，这在以后成为了很多生成式模型的基础。

同时判别器也借由集成学习的方法，使用了多个判别器联合判断。

最后的Loss 也是一个单纯的对抗损失，这个对抗损失有着改良的空间，同时也有新加入其他loss的余地。

这里最大吐槽的是用的全是线性层，可能当时卷积层好不够流行（？），这个问题将被以后解决。

综上所述，这篇文章真的提出了很新颖的概念，美滋滋。
